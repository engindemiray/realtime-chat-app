## Languages & Tools

- JavaScript
- Node.js
- Express.js
- Socket.io

## Installation
Run these commands in Terminal.

Creating a package.json file:
``` bash
$ npm init --yes
```
To install Express & Socket.io:
``` bash
$ npm i express socket.io
```
To start the app:
``` bash
$ nodemon server.js
```
